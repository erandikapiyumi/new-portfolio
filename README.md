# 🚀 Modern Developer Portfolio

Welcome to my personal portfolio website! A sleek, modern, and fully responsive portfolio designed with a Cyberpunk-inspired dark theme, dynamic animations, and a glassmorphism UI.

## 🌐 Live Demo

See it in action here:
*   **GitHub Pages:** [erandikapiyumi.github.io/new-portfolio](https://erandikapiyumi.github.io/new-portfolio/)
*   **Vercel Mirror:** [new-portfolio-ngqr.vercel.app](https://new-portfolio-ngqr.vercel.app/)

---

## ✨ Key Features

*   **Cyberpunk Glow Aesthetics:** Beautiful dark mode with dynamic cyan glow effects.
*   **Smooth Animations:** Bouncing profile avatar and interactive scale-up hover effects.
*   **Modern Glassmorphism:** UI components built with sleek backdrop blur filters.
*   **Responsive Grid Layout:** Perfectly optimized for Mobile, Tablet, and Desktop screens.
*   **Dynamic Skills Section:** Showcasing frontend and backend tech stacks with high-quality icons.

---

## 🛠️ Tech Stack & Tools

*   **Frontend Framework:** React.js (Vite)
*   **Styling:** Modern CSS3 (Grid, Flexbox, & Custom Animations)
*   **Icons:** Devicons
